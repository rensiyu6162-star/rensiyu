始终交付可运行的 HTML 文件，并在写完后主动用 `open` 命令在浏览器里打开，不要等用户问"怎么预览"。

**Why:** 用户是代码小白，不知道怎么预览 HTML 文件，每次都需要问。主动打开可以节省来回。

**How to apply:** 每次写完或修改完 HTML 文件后，立即执行 `open <file>` 打开浏览器预览，无需等用户要求。
